Spotify adblock.
App mutes spotify's adverisement.

```
usage: python3 advmute.py [-h] [-o LOG_FILE_NAME]
or daemon: advmute.py [-h] [-o LOG_FILE_NAME] &

optional arguments:
  -h, --help            show this help message and exit
  -o LOG_FILE_NAME, --out LOG_FILE_NAME
                        log file
```
